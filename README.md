## Description

This C++ calculator project provides basic arithmetic operations, including addition, subtraction, multiplication, and division. It serves as an educational example for beginners learning C++ programming.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Requirements

- C++ compiler (e.g., g++)
